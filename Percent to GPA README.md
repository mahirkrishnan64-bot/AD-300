Percent to GPA Converter
This Java program accepts a student's percentage score and converts it to a GPA based on the grading scale provided in the assignment.

Project description
The program uses an if-else if ladder to compare the entered percent against the grading scheme and then prints the correct GPA value. Java commonly uses if-else statements for ordered numeric comparisons like grade boundaries.


- Accepts a student's percent as input

- Converts the percent to the correct GPA

- Rejects invalid input below 0 or above 100

- Follows the assignment grading table from 4.0 to 0.0


Sample runs

Enter student's percent: 96.5
GPA: 4.0

Enter student's percent: 84.2
GPA: 2.9

Enter student's percent: 66.7
GPA: 1.1


Edge cases

100 -> GPA: 4.0

65.0 -> GPA: 1.0

101 -> Invalid percent. Enter a value from 0-100
